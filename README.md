## بناء ملف EXE

يحوّل البرنامج لملف تنفيذي مستقل **لا يحتاج Python** على الجهاز المستهدف.

### خطوات البناء:
```
1. افتح مجلد المشروع
2. انقر مرتين على:  build.bat
3. انتظر حتى ينتهي (3-10 دقائق حسب الجهاز)
4. النتيجة في مجلد:  dist\صيدلية\
```

### محتوى مجلد dist بعد البناء:
```
dist\
└── صيدلية\
    ├── صيدلية.exe        ← الملف التنفيذي الرئيسي
    ├── assets\           ← الخطوط والإعدادات
    ├── database\         ← schema قاعدة البيانات
    └── [مكتبات أخرى]
```

> **ملاحظة:** مجلد `dist\صيدلية\` كامل يجب أن يُنقل معاً — لا يعمل الـ exe وحده.

## إنشاء ملف Setup

يحوّل مجلد `dist\صيدلية\` إلى ملف `pharmacy_setup_v1.0.0.exe` يُثبَّت بنقرة واحدة.

### المتطلب: تثبيت Inno Setup
```
1. حمّل Inno Setup 6 من: https://jrsoftware.org/isdl.php
2. ثبّته على جهازك
```

### خطوات إنشاء Setup:
```
1. تأكد من وجود مجلد dist\صيدلية\ (نتيجة build.bat)
2. افتح Inno Setup Compiler
3. File → Open → اختر ملف:  installer.iss
4. اضغط F9 أو Build → Compile
5. ملف Setup سيُنشأ في:  output\pharmacy_setup_v1.0.0.exe
```

### توزيع البرنامج:
```
انقل ملف:  output\pharmacy_setup_v1.0.0.exe
إلى أي جهاز Windows 10/11 (64-bit) وشغّله كمسؤول (Run as Administrator)
```

## أول تشغيل

عند أول تشغيل، يُنشئ البرنامج تلقائياً:
- قاعدة بيانات `pharmacy.db`
- مستخدم افتراضي (Super Admin)
- الأدوار والصلاحيات الافتراضية
- مجلد `receipts/` للفواتير

### بيانات الدخول الافتراضية:
```
اسم المستخدم:  admin
كلمة المرور:   [تم تعيينها أثناء التثبيت الأول]
```
> ⚠️ **مهم:** كلمة المرور الافتراضية للحساب الأول هي `admin123`  
> غيّرها فور تسجيل الدخول: الإعدادات ← تغيير كلمة المرور  
> كلمات المرور مشفّرة بـ bcrypt ولا تُحفظ بأي صورة قابلة للقراءة

### أول إعداد:
```
1. سجّل الدخول
2. اذهب لـ الإعدادات ← معلومات الصيدلية
3. أدخل: اسم الصيدلية، العنوان، الهاتف، العملة
4. اضغط "حفظ"
```

## workflow البيع

```
نقطة البيع (POS)
   ↓ امسح الباركود أو ابحث بالاسم
   ↓ أضف للسلة
   ↓ اختر: إرسال للكاشير [Shift+S] أو إتمام مباشر [Shift+W]

الكاشير (لو أرسلت من POS)
   ↓ الفاتورة تظهر تلقائياً (تحديث كل 3 ثوانٍ)
   ↓ أدخل المبلغ المدفوع
   ↓ اضغط "إتمام البيع"
   ↓ طباعة الفاتورة PDF (اختياري)
```

**ملاحظة مهمة:** المخزون **لا يُخصم** إلا عند إتمام الدفع الفعلي — وليس عند الإرسال للكاشير.

## حل المشاكل الشائعة

### ❌ البرنامج لا يفتح بعد التثبيت
```
الحل:
1. تأكد من تشغيله بصلاحيات المسؤول (Right Click → Run as Administrator)
2. تأكد أن Windows Defender لا يحجبه
3. أضفه للاستثناءات: Windows Security ← Virus & threat protection → Exclusions
```

### ❌ الفواتير بها أحرف عربية غير متصلة
```
الحل:
تأكد من وجود ملف:  assets\Amiri-Regular.ttf
لو مفقود حمّله من: https://fonts.google.com/specimen/Amiri
```

### ❌ خطأ عند فتح شاشة معينة
```
الحل:
افتح CMD وشغّل: python main.py
ستظهر رسالة الخطأ الكاملة — أرسلها للدعم الفني
```

### ❌ قاعدة البيانات تالفة
```
الحل:
1. أغلق البرنامج
2. احذف ملف pharmacy.db
3. شغّل البرنامج من جديد (سيُنشئ قاعدة بيانات جديدة)
4. استعد النسخة الاحتياطية من: الإعدادات ← النسخ الاحتياطي
```

### ❌ الشاشة تبدو صغيرة أو عناصرها مضغوطة
```
الحل:
1. Right Click على الملف التنفيذي ← Properties
2. Compatibility ← Change high DPI settings
3. فعّل "Override high DPI scaling behavior"
4. اختر "Application" من القائمة
```
